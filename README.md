# WordSteps 阶梯背单词

离线优先的阶梯式英语词汇学习应用。按难度阶梯组织词库，支持「增量模式」只学新词，内置离线英文释义与发音包，打开网页即可使用，无需联网。

## 功能特性

- **阶梯词库**：850 基础词 → 初中 → 高考 → 四级 → 六级 → 考研 → 雅思 → SAT → 托福，由易到难。
- **增量模式**：开启后，除 850 基础词与初中词外，每本词书只显示与「之前所有词书并集」的差集新词，避免重复背诵。
- **离线英文释义**：内置 `books/en_defs.js`（约 1.4 万词），加载即就绪，零网络依赖。
- **音标与发音**：词卡显示英/美音标，点击即可播放本地发音。
- **品牌化界面**：WordSteps 阶梯背单词，卡片式学习体验。

## 本地运行

直接用浏览器打开 `index.html` 即可（建议通过本地静态服务器以避免某些浏览器的 file:// 限制）：

```bash
# 任选其一，在仓库根目录执行后访问 http://localhost:8000
python -m http.server 8000
```

## 数据更新

英文释义包由构建脚本生成，可随版本重新抓取并更新：

```bash
# 全量构建（含词库与英文释义包，支持断点续传）
python tools/rebuild_v3.py

# 仅补抓缺失的英文释义
python tools/resume_en_defs.py
```

词库原始数据来自各上游开源词表，构建逻辑见 `tools/rebuild_v3.py`。

## 目录结构
